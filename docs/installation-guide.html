RAVE Documentation

Chapter 2 Installation Guide

RAVE is written as an “R” package. To properly install RAVE, please carefully follow the instructions step-by-step. We found 90% installation issues are due to not following the instructions (for example, system prerequisites).

Hardware Requirements

  • Minimal requirement: 4 CPU cores with 8 GB RAM
  • Recommended: 8+ CPU with 32+ GB RAM

System Library Prerequisites

  • MacOS
  • Windows
  • Debian Linux (Ubuntu)
